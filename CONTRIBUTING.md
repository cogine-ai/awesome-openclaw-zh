# 贡献指南

欢迎你把真实可用的 OpenClaw 用法补充进来。

## 投稿标准（简单版）

- 真实用过，不是纯想法
- 非技术用户看得懂
- 提供可直接复制的提示词
- 明确写清楚“能干什么”

## 提交流程

1. 复制 [usecases/TEMPLATE.md](./usecases/TEMPLATE.md)
2. 放到对应分类目录（`usecases/*`）
3. 在 [resources/usecases-index.md](./resources/usecases-index.md) 对应分类加一行
4. 发起 PR

## 命名建议

- 文件名统一英文短横线：`xxx-xxx.md`
- 尽量用动作导向命名：`daily-brief`、`inbox-cleanup`、`meeting-notes`

## 不建议内容

- 纯广告内容
- 无法复现的截图贴
- 明显违规或高风险自动化（如未授权数据抓取）
