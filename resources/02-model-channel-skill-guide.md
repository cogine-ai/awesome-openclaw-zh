# 模型、渠道、技能怎么选（实用版）

## 模型怎么选

- 日常任务：用平衡型模型
- 重推理任务：临时切到高性能模型
- 高频巡检：用便宜模型

关键不是“最强模型”，而是“模型分工”。

## 渠道怎么选

- 个人用户：Telegram / 手机消息渠道
- 中文团队：飞书 / 企业微信
- 开发协作：Discord / Slack

## 技能怎么装

- 先装 3 个高频技能，再扩展
- 每新增技能都要看权限范围
- 不要一次装太多，避免定位问题困难

## 推荐先做的 3 个组合

1. 晨间简报 + 日历提醒 + 邮件整理
2. 内容选题 + 社交发布 + 阅读清单
3. 任务追踪 + 会议纪要 + 知识库检索
