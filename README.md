# OpenClaw 中文案例精选（awesome-openclaw-zh）

一个面向中文用户的 OpenClaw 实战仓库：
**先 5 分钟用起来，再不断扩充能力边界。**

本仓库主打：
- 非技术用户友好
- 案例覆盖广（已收录 163 个）
- 复制提示词即可上手
- 以真实社区用法为核心
- 每个案例都包含来源对齐信息（场景/动作/工具/边界）
