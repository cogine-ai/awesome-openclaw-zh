# Awesome OpenClaw 中文精选（awesome-openclaw-zh）

一个面向中文用户的 OpenClaw 实战仓库：
**先看懂能干什么，再 5 分钟用起来。**

本仓库主打：
- 非技术用户友好
- 案例覆盖广（已收录 99 个）
- 复制提示词即可上手
- 以真实社区用法为核心
