# Alexa CLI 自然语言控制（含智能家居设备）

> 通过 Alexa CLI 把语音意图转为设备控制动作，接入 OpenClaw 自动化链路。

## 这个案例能帮你做什么

- 把 Alexa 生态设备纳入 OpenClaw 可调用范围。
- 支持自然语言控制，不必手写复杂指令。
- 为家庭自动化补齐“语音入口 + 规则执行”闭环。

## 你需要的 Skills（按类型）

| 类型 | Skill / 工具 | 用途 | 来源 |
|---|---|---|---|
| 外部 | `Alexa CLI` | 控制 Echo 与 Alexa 设备 | Alexa CLI |
| 内置 | `OpenClaw 指令编排` | 把自然语言映射为设备动作 | OpenClaw Built-in |

## 快速体验版（先跑一轮）

1. 先连接 1 个可控设备（例如灯或插座）。
2. 跑 3 个自然语言指令（开、关、状态查询）。
3. 核对实际设备反馈。

```text
请把以下自然语言指令映射为 Alexa 控制动作：
- 打开客厅灯
- 关闭客厅灯
- 查询客厅灯当前状态
输出执行结果和失败原因。
```

## 稳定自动版（可长期运行）

### 推荐流程

1. 先建立设备命名规范。
2. 将高频语音场景做成固定快捷语义模板。
3. 关键场景加入失败重试和状态回读。

## 成功标准

- [ ] 常见设备可被稳定控制。
- [ ] 自然语言指令理解一致。
- [ ] 执行后有明确状态回执。
