# 车价谈判自动化（多经销商协商）

> 用 OpenClaw 并行沟通多家经销商，持续压价并比较条件。

## 这个案例能帮你做什么

- 把“逐个聊天砍价”变成并行谈判流程。
- 统一比较报价、附加条件和时间窗口。
- 降低人工沟通成本，提高议价效率。

## 你需要的 Skills（按类型）

| 类型 | Skill / 工具 | 用途 | 来源 |
|---|---|---|---|
| 内置 | `浏览器 / 邮件 / iMessage` | 多通道谈判触达 | OpenClaw Built-in |
| 内置 | `OpenClaw 任务编排` | 跟进报价与回合策略 | OpenClaw Built-in |

## 快速体验版（先跑一轮）

1. 准备 3 家经销商与同一配置需求。
2. 让 OpenClaw 生成统一询价模板并发起第一轮沟通。
3. 收敛成对比表（裸车价、金融条件、赠品、交付周期）。

```text
请帮我做一轮标准化询价：
- 目标车型与配置保持一致
- 输出每家经销商的可比信息表
- 标注下一轮谈判的优先顺序与目标降幅
```

## 稳定自动版（可长期运行）

### 推荐流程

1. 按“报价完整度 + 总成本 + 可谈空间”自动排序。
2. 对高潜力商家做二轮压价。
3. 只在关键节点（签约前）触发人工确认。

## 成功标准

- [ ] 经销商报价可横向比较且字段统一。
- [ ] 谈判回合有记录，策略可复盘。
- [ ] 最终决策前保留人工确认。
