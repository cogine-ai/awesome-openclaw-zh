# Garmin 运动数据热力图（个人轨迹可视化）

> 把 Garmin 运动数据导入后，用 OpenClaw 快速生成可交互热力图。

## 这个案例能帮你做什么

- 直观看到你的运动分布与活跃区域。
- 支持按范围查看不同城市/阶段的跑步记录。
- 为训练复盘提供空间维度的数据视角。

## 你需要的 Skills（按类型）

| 类型 | Skill / 工具 | 用途 | 来源 |
|---|---|---|---|
| 外部 | `Garmin 数据` | 运动轨迹输入 | Garmin |
| 外部 | `地图 API` | 生成可交互热力图 | Map API |
| 内置 | `OpenClaw 对话式迭代` | 用少量提示优化图表交互 | OpenClaw Built-in |

## 快速体验版（先跑一轮）

1. 先导入一段时间的 Garmin 数据。
2. 生成第一版热力图。
3. 追加第二次优化（导航、滑块、指标卡）。

```text
请基于已导入的 Garmin 运动数据生成热力图：
- 支持缩放和时间筛选
- 显示累计活动分布
- 输出可继续优化的交互建议
```

## 稳定自动版（可长期运行）

### 推荐流程

1. 周期同步最新运动数据。
2. 自动更新地图可视化。
3. 月度输出训练分布变化摘要。

## 成功标准

- [ ] 热力图可稳定反映运动轨迹。
- [ ] 支持按时间或区域筛选。
- [ ] 可视化可用于复盘与计划调整。
