# OpenClaw macOS 菜单栏控制台

> 用一个菜单栏小应用管理 OpenClaw 网关状态、日志和告警。

## 这个案例能帮你做什么

- 在桌面常驻入口查看 OpenClaw 关键状态。
- 一键执行启动/停止/重启等操作。
- 通过通知机制及时感知状态变化。

## 你需要的 Skills（按类型）

| 类型 | Skill / 工具 | 用途 | 来源 |
|---|---|---|---|
| 外部 | `macOS Menu Bar App` | 状态展示与快捷控制 | macOS |
| 内置 | `Gateway 状态与日志读取` | 监控运行健康度 | OpenClaw Built-in |
| 外部 | `桌面通知` | 状态变化提醒 | macOS Notification |

## 快速体验版（先跑一轮）

1. 先做只读版本：显示网关在线状态与最近日志。
2. 再加“重启”单按钮。
3. 最后补状态变化通知。

```text
请先生成一个最小菜单栏版本：
- 显示 OpenClaw 当前状态
- 展示最近日志摘要
- 提供重启入口（执行前确认）
```

## 稳定自动版（可长期运行）

### 推荐流程

1. 状态读取与控制动作分离。
2. 高风险动作前增加确认弹窗。
3. 异常日志触发通知并附带定位信息。

## 成功标准

- [ ] 菜单栏可稳定展示实时状态。
- [ ] 常用控制动作可一键触发。
- [ ] 异常状态会及时通知。
