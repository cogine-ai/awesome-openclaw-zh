# 用例标题（例：每日晨间简报）

> 一句话说明这个案例能帮你做什么。

## 你可以拿它做什么

用 2-3 句话说明实际收益。

## 3 步快速开始

1. 打开 OpenClaw（Web 或聊天渠道）
2. 复制下面提示词发给 OpenClaw
3. 替换方括号里的个人信息

## 复制即用提示词

```text
你是我的 OpenClaw 助手，请帮我完成「[用例标题]」。

目标：[写清楚这个任务的结果]

请按这个结构执行：
1. 先告诉我今天要做什么
2. 再给出直接可执行步骤
3. 需要我确认的项放在最后

我的背景：
- 城市/时区：[你的城市]
- 常用渠道：[微信/飞书/Telegram/邮箱]
- 重点关注：[你的主题]
```
