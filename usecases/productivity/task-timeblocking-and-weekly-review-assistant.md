# 任务时间块排程与周复盘助理

> 先帮你判断什么最重要，再把任务排进日历，并把会议转录变成每周复盘和后续动作。

## 这个案例能帮你做什么

- 按重要性和紧急性给任务打分，再把高优先级事项排进日历 time blocks。
- 基于会议转录与笔记输出 weekly review，而不是只靠回忆复盘。
- 继续向外扩展到会前人物 briefing、家庭学校提醒、工作发票汇总等辅助任务。

## 快速体验版（先跑一轮）

1. 先拿一周任务清单、日历和最近几场会议记录做预演。
2. 本轮先输出排程建议和 weekly review，不直接写入日历。
3. 如果效果满意，再把会前 briefing、学校提醒、发票整理接进来。

```text
你是我的 OpenClaw 助手。
请先帮我做“任务时间块排程与周复盘助理”的一轮预演：
1. 根据我的任务列表，按重要性和紧急性评分。
2. 把最高优先级任务排进本周日历，输出建议的 time blocks 和可能冲突。
3. 根据我的会议转录与笔记，生成一份 weekly review：本周推进了什么、卡点是什么、下周重点是什么。
4. 另外列出：需要会前 briefing 的对象、需要提醒的家庭学校事项、需要汇总的工作发票。
5. 本轮先只输出方案，不直接改日历或发通知。
```

## 稳定自动版（可长期运行）

### 推荐流程

1. 每天先对任务做一次重要性与紧急性排序，再把最重要的事项排进日历。
2. 自动检查日历冲突，并对无法容纳的任务重新建议时间块。
3. 每周固定读会议转录与笔记，输出一份 weekly review，帮助你决定下周主线。
4. 需要见人前，先做人物 research 和 briefing docs；需要处理家庭事项时，再补考试提醒。
5. 月度或结算前，再让 OpenClaw 汇总发票和工作摘要，形成可交付版本。

## 成功标准

- [ ] 高优先级任务能被稳定排进日历，而不是只停留在待办列表里。
- [ ] weekly review 明确引用会议转录和笔记信息。
- [ ] 日历冲突、会前 briefing、家庭提醒和工作汇总可以逐步接入同一套流程。
