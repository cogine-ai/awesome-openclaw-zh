# 批量文件处理助手

> 批量处理文档并生成摘要，适合资料密集型任务。

## 你可以拿它做什么

这个案例重点是：**批量处理文档并生成摘要，适合资料密集型任务。**

适合希望“少折腾、快见效”的用户。

## 3 步快速开始

1. 打开 OpenClaw（建议先连接一个常用聊天渠道）。
2. 复制下面提示词并发给 OpenClaw。
3. 按你的场景替换方括号内容，先跑通一次再逐步优化。

## 复制即用提示词

```text
请帮我执行「批量文件处理助手」。

目标：批量处理文档并生成摘要，适合资料密集型任务。

输出要求：
1. 先给我一个今日执行版本（可直接用）
2. 再给我一个每周复用版本（可持续）
3. 如果有风险或需要确认，放在最后单独列出

我的信息：
- 使用场景：[个人/团队]
- 常用渠道：[微信/飞书/Telegram/邮箱]
- 时间偏好：[如每天9点]
```

## 适合人群

- 希望快速落地 OpenClaw 的非技术用户
- 想把重复事项交给自动化的内容/运营/产品用户
- 想建立长期可复用流程的小团队
