# 阅读材料重打包成极简 HTML/CSS 阅读器

> 把原本不适合长时间阅读的资料，重排成一个更轻、更顺手的 HTML/CSS 阅读界面。

## 这个案例能帮你做什么

- 把“内容能看”提升到“内容更愿意看、能更快看完”。
- 根据你的阅读习惯重做版式，而不是被原始页面样式绑住。
- 额外加入 adjustable sizing 和 speed-read mode，让资料更适合快速消化。

## 快速体验版（先跑一轮）

1. 先拿一份你最近真想读完的资料做预演。
2. 本轮重点只放在阅读体验，不做额外功能堆叠。
3. 输出一个能直接打开的 HTML/CSS 版本即可。

```text
你是我的 OpenClaw 助手。
请把这份阅读资料重打包成一个极简 HTML/CSS 阅读器：
1. 保留核心内容，但把版式改成更适合长时间阅读。
2. 提供 adjustable sizing。
3. 提供 speed-read mode。
4. 输出一个我可以直接打开使用的轻量 HTML/CSS 文件。
```

## 稳定自动版（可长期运行）

### 推荐流程

1. 每次拿到新的长文、手册或资料时，先判断原始版式是否真的适合阅读。
2. 如果不适合，就把内容重打包成一个更轻的 HTML/CSS 单页版本。
3. 保留 adjustable sizing 和 speed-read mode，让它适配不同阅读节奏。
4. 最终交付尽量保持为简单、可直接打开的文件，而不是再引入复杂依赖。

## 成功标准

- [ ] 阅读材料能被稳定重排成可直接打开的 HTML/CSS 文件。
- [ ] adjustable sizing 和 speed-read mode 可用。
- [ ] 重打包后的版本比原始阅读体验更轻、更快进入状态。
